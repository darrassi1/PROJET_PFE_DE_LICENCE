############Gestion d'Application avec Symfony 5.0.11
Description
Ceci est un projet de gestion d'application développé en utilisant Symfony 5.0.11. L'application offre des fonctionnalités pour gérer des clients, des fournisseurs, des commandes de clients, et permet également la création de nouveaux médicaments. Il s'appuie sur plusieurs composants Symfony tels que Doctrine pour la gestion de la base de données et Twig pour les vues.

Fonctionnalités clés
Gestion des clients : Ajout, modification et suppression de clients.
Gestion des fournisseurs : Suivi des informations sur les fournisseurs.
Commandes de clients : Enregistrement et suivi des commandes passées par les clients.
Création de nouveaux médicaments : Possibilité d'ajouter de nouveaux médicaments au système.
Prérequis
Avant de déployer ce projet, assurez-vous d'avoir les éléments suivants installés :

Symfony 5.0.11
Composer
Doctrine (ORM)
Twig (moteur de template)
Installation
Clonez ce référentiel sur votre machine locale.
Exécutez composer install pour installer les dépendances.
Configurez votre base de données dans le fichier .env et exécutez les migrations Doctrine.
Utilisation
Démarrez le serveur Symfony avec la commande symfony server:start.
Accédez à l'application dans votre navigateur en utilisant l'URL fournie par Symfony.
