                                                                        Gestion d'Application avec Symfony 5.0.11

Fonctionnalités clés
Gestion des clients : Ajout, modification et suppression de clients.
Gestion des fournisseurs : Suivi des informations sur les fournisseurs.
Commandes de clients : Enregistrement et suivi des commandes passées par les clients.
Création de nouveaux médicaments : Possibilité d'ajouter de nouveaux médicaments au système.
